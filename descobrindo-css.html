Descobrindo propriedades CSS

Propriedades CSS pesquisadas e usadas por nós no nosso dia a dia, porém não foram
passadas em aula ainda.

cursor

Propriedade utilizada para alterar o visual do cursor do usuário, podendo ser usado para mostrar alguma estado atual da sua aplicação ou possibilidade de interação do usuário

white-space

Propriedade CSS para lidar com os espaços em branco de um elemento.

Normal Texto de teste utilizado para preencher espaço para poder mostrar como o text-overflow funciona na prática.
No-Wrap Texto de teste utilizado para preencher espaço para poder mostrar como o text-overflow funciona na prática.

overflow

Define o comportamento de um elemento pai caso o seus elementos filhos ultrapassem o seu tamanho.

Visible Texto de teste utilizado para preencher espaço para poder mostrar como o overflow funciona na prática.
Hidden Texto de teste utilizado para preencher espaço para poder mostrar como o overflow funciona na prática, em uma experiência de mundo real
Scroll Texto de teste utilizado para preencher espaço para poder mostrar como o overflow funciona na prática, em uma experiência de mundo real

text-overflow

Propriedade CSS que define como um texto irá se comportar ao ultrapassar o seu espaço pré-estabelecido pelo seu elemento pai.

Vale lembrar que essa propriedade precisa estar seguida de um white-space: nowrap e um overflow: hidden

clip Texto de teste utilizado para preencher espaço para poder mostrar como o white-space funciona na prática.
ellipsis Texto de teste utilizado para preencher espaço para poder mostrar como o white-space funciona na prática.
