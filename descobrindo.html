Descobrindo tags

Tags pesquisadas e usadas por nós no nosso dia a dia, porém não foram
passadas em aula ainda.

<select> </select>

O elemento HTML select (<select>) representa um controle que apresenta um menu de opções.
As opções dentro do menu são representadas pelo elemento <option>, que podem ser agrupados por elementos <optgroup>.
As opções podem ser pré-selecionadas para o usuário.

<option> </option>

Em um formulário Web, o elemento HTML <option>
é usado para criar um controle que representa um item dentro de um elemento HTML5 <select>, <optgroup> ou <datalist>.

<optgroup> </optgroup>

Em um Formulário Web, o elemento HTML <optgroup> cria um agrupamento de opções dentro do elemento<select>.

<details> </details>

O elemento HTML details (<details>) é usado como uma ferramenta de onde o usuário irá obter informações adicionais.
Ele abre um espaço, aonde como título deve-se user a tag <summary> para informar o "detalhe" que vai ser mostrado.
